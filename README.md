# Interview Experience Sharing Platform

A full-stack web application built using Flask, SQLite, HTML, CSS, and JavaScript that allows users to share and explore interview experiences from different companies and roles.

## Features

- JWT Authentication
- Add, Edit, Delete Experiences
- Upvote System
- Live Search
- Company Filtering
- Dark Mode
- Responsive Dashboard UI
- User-based Posting System

## Tech Stack

- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
- JavaScript
- JWT Authentication

## Project Purpose

This platform helps students and job seekers prepare for interviews by accessing real interview experiences shared by users.
